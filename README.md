# 📚 AI Study Planner

An AI-powered Study Planner developed using **HTML, CSS, JavaScript, Local Storage, and Dify AI Workflow** to help students organize study tasks, generate personalized study schedules, and improve exam preparation.

---

## 🚀 Features

* ➕ Add study tasks
* ✏️ Edit existing tasks
* 🗑️ Delete tasks
* ✅ Mark tasks as completed
* 📊 Dynamic progress tracker
* 💾 Local Storage support (data persists after page refresh)
* 📋 Priority-based task management (High, Medium, Low)
* 🤖 AI-generated personalized study plans using Dify Workflow
* 📱 Responsive user interface

---

## 🛠️ Technologies Used

* HTML5

* JavaScript (ES6)
* Local Storage API

## 🤖 Dify AI Integration

This project integrates with a **Dify Workflow** to generate intelligent study plans.

### Setup Steps

1. Create a Workflow in Dify.
2. Publish the workflow.
3. Obtain your Workflow API URL and API Key.
4. Configure them in the JavaScript code or (recommended) through a backend server.
5. Run the application.

---

## 📌 How It Works

1. Add study tasks with subject, topic, study hours, and priority.
2. Tasks are stored in JavaScript arrays and saved using Local Storage.
3. The application displays all tasks and tracks completion progress.
4. When the AI feature is used, the task details are sent to the Dify Workflow.
5. The AI returns a personalized study schedule, revision strategy, and productivity recommendations.
